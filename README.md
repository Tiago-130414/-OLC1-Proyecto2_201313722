# -OLC1-Proyecto2_201313722
Proyecto 2 - Compiladores 1 
El siguiente proyecto es un verificador de copias en lenguaje java, el cual se basa en un analizador lexico y sintactico descendente
se realizo utilizando una gramatica por la izquierda el cual verificaba la sintaxis de sentencias y ciclos como lo son el if,
switch, for, while, do while.Tambien verifica la sintaxis de impresiones en consola asi como importaciones, declaraciones y asignaciones 
de variables.Cabe resaltar que todo este tipo de instrucciones deben ir dentro de una clase y un metodo. Este proyecto fue elaborado con
el lenguaje javascript, go, respaldados con un servidor en node js y un cliente en go.
